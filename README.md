# arvore binária sem balanceamento em javascript

Algoritmo feito com a versão LTS do node.js e ECMA6

Para compilar e testar abra o terminal ou cmd

Navegar ate o diretório digitar pra executar

node script.js > output.js

O node.js executará o algoritmo usando o input.txt
e colocará no output.txt a saída

Para quem usar o linux deixei comentado a forma de ler 
o arquivo de entrada
